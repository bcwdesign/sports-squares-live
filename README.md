# ClutchSquares — Real-Time Watch Party Game for Live Sports

## Overview

ClutchSquares is a multiplayer web app that brings the classic sports squares game into a real-time, social watch-party experience. Players can join from their phones, claim squares, and watch winners update live as scores change.

## What We Built During Blackathon

During Blackathon 2026, we built the core real-time watch-party experience, including:

- Live Watch Party Overlay (TV / second-screen experience)

- Real-time winner detection and highlight system

- Winner Celebration Engine (animated square + player spotlight + confetti)

- QR-based instant join flow for frictionless onboarding

- Real-time synchronization improvements across players and host

## Features

- Create and host live games

- Invite players via link or QR code

- Claim squares on a 10x10 board

- Randomized score axes

- Real-time score updates

- Automatic winner calculation

- Live chat between players

- TV overlay for shared viewing

## Tech Stack

- Lovable (frontend + backend scaffolding)

- Supabase (authentication, database, realtime)

- WebSockets / realtime channels

- CSS animations for real-time visual effects

## How It Works

1. Host creates a game

2. Players join via link or QR code

3. Players claim squares on a 10x10 board

4. Host locks and randomizes the board

5. Scores update in real time

6. Winners are automatically calculated and highlighted
